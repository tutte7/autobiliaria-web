# Fuentes personalizadas

## Square 721 TL Bold Extended

Para usar la fuente "Square 721 TL Bold Extended", coloca el archivo de fuente en esta carpeta con uno de estos nombres:

- `Square721BoldExtended.ttf` (formato TTF)
- `Square721BoldExtended.otf` (formato OTF)

El sistema buscará ambos formatos automáticamente. Si tienes el archivo con otro nombre, puedes renombrarlo o actualizar la configuración en `app/layout.tsx`.

### Pasos:

1. Descarga o copia el archivo de la fuente "Square 721 TL Bold Extended"
2. Colócalo en esta carpeta (`fonts/`)
3. Renómbralo a `Square721BoldExtended.ttf` o `Square721BoldExtended.otf`
4. La fuente se aplicará automáticamente al texto "Autobiliaria" en el navbar
